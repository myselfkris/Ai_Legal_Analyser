# Ai_Legal_Analyser

A Generative AI-powered Legal Document Analyzer that helps users parse, analyze, and understand complex legal contracts.

## Features
- **PDF Parsing**: Extract text from legal documents.
- **AI Analysis**: Use GenAI to identify key terms, risks, and obligations.
- **Interactive Interface**: Easy-to-use dashboard for legal professionals.

## Tech Stack
- **Backend**: FastAPI, Python
- **AI**: Google Gemini API
